Printer Test

⚠️ Important: When the print dialog opens, look for "Paper Size" and select "Receipt" or "80mm"!

How to Test:

  1. Connect thermal printer via USB
  2. Click a test button below
  3. In the print dialog, find "Paper Size" or "Printer Properties"
  4. Select "Receipt" or "80mm" (NOT A4 or Letter!)
  5. Set margins to "None" or "Minimal"
  6. Click Print
